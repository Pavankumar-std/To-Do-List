
# To-Do List

## Description
This is a simple To-Do List application developed using Python. It helps users manage their daily tasks using a command-line interface.

## Features
- Add a new task
- View all tasks
- Delete a task
- Exit the program

## Technologies Used
- Python

## How to Run

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python todo.py
```
